## math/rand
Позволяет с помощью инструмента *Intn("значение")* генерировать рандомное целое положительное число до заданного значения,тоесть если у нас Intn(5),  то генерится число от 0 до 4

Пример:
```go
package main
import (
	"fmt"
	"math/rand"// импортируем нужную библиотеку
)
func main() {
	fmt.Println("Рандомная цифра", rand.Intn(5)) // генерим
}
```

подробнее тут: https://pkg.go.dev/math/rand

## time
Отсюда мы рассматриваем инструмент Sleep("значение"),который позволяет на некоторое время приостановить прогу

Кстати значение по сути складывается из (n * time."Величина"),где n - любое целое неотрицательное число

Пример:
```go
for i := 1; i <= 5; i++ {
	fmt.Println(i)
	time.Sleep(500 * time.Millisecond) // приостановка на 500мс
}
```

Подробнее: https://pkg.go.dev/time
