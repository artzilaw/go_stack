Чтобы была возможность не только в проге код править, но и вводить во время выполнения программы какие-то данные, надо уметь пользоваться пользовательским вводом
Пояснять буду на примере задания:
```
// Введите команду: добавить лук
// вы хотите добавить лук

// Введите команду: удалить морковь
// вы кажется хотите удалить морковь

// неивестная команда
// |
// help
// вот список команд!

// выйти
```
В Go есть пакет bufio, через него можно создать сканер - нечто,что читает поток входящей инфоромации,но как понять что за именно поток информации он будет читать? ему для этого надо передать os.Stdin - входной поток данных из консоли:
```go
package main

import (
	"bufio"
	"os"
)

func main() {
	bufio.NewScanner((os.Stdin))
}
```

 Выше у нас просто демонстрация сканера,который читает поток из консоли,но вообще, нам еще надо,чтобы на время ввода блокировалось выполнение команды, с этим нам поможет Scan(), который возвращает bool, мол true, если пользовательский ввод бы успешно прочитан, иначе false
 ```go
 package main

import (
	"bufio"
	"fmt"
	"os"
)

func main() {
	scanner := bufio.NewScanner((os.Stdin))
	fmt.Println("Введите команду")
	scanner.Scan()
}
 ```
При желании можно  и так оформить:
```go
package main

import (
	"bufio"
	"fmt"
	"os"
)

func main() {
	scanner := bufio.NewScanner((os.Stdin))
	fmt.Println("Введите команду")
	ok := scanner.Scan()
	if !ok {
		fmt.Println("Ошибка ввода")
		return
	}
}
```

При желании можно конструкцию с ветвлением и ok вот так сократить, у нас тут сразу считывается ввод, результат кладется в ok и потом проверяется
```go
if ok := scanner.Scan(); !ok {
	fmt.Println("Ошибка ввода")
	return
}
```

Чтобы вернуть строку, которую ввел пользователь, нужнно использовать Text()
```go
text := scanner.Text()
fmt.Println(text)
```

## Разбиение длинной строки на массив

Допустим пользователь вводит команду, после которой идет значение, которое можно передать, это можно реализовать через пакет strings:
```go
text := scanner.Text()
strings.Fields(text)
Ввод:
добавить вентилятор тарелку вилку
Вывод:
[добавить вентилятор тарелку вилку]
```
Метод fields принимает строку и разбивает ее на массив, следовательно, мы можем просто считывать 0 элемент массива как команду,а остальные "слова",как значения 

Далее, присваиваем cmd значение `fields[0]` и смотрим, что за это команда, что она делает, ну или это не команда вовсе
```go
fmt.Println("Команда:", fields[0])
cmd := fields[0]
if cmd == "добавить" {
	fmt.Println("Вы хотите добавить:")
} else if cmd == "удалить" {
	fmt.Println("Вы кажется хотите удалить что-то?")
} else if cmd == "help" {
	fmt.Println("Команда: добавить {добавить что-от}")
	fmt.Println("-- позволяет что-то добавлять")
	fmt.Println("Команда: удалить {удалить что-от}")
	fmt.Println("-- позволяет что-то удалить")
} else {
	fmt.Println("Вы ввели неизвестную команду")
}
```


Далее сделаем пару исправлений,а именно Напишем в первом выводе Print(), а не Println(), чтобы текст программы и наш были на одной строке, ну и добавим обработку ввода пустой строки,чтобы у нас не выскакивало ошибок
```go
fmt.Print("Введите команду: ")
// Код
fields := strings.Fields(text)
if len(fields) == 0 {
	fmt.Println("Вы ничего не ввели")
	return
}
```
Также было бы хорошо прописать, что конкретно мы хотим удалить или добавить, для этого в теле нужного нам ветвления(куда мы попадаем, если команда верная), инициализируем пустую строку и цикл, в теле цикла надо добавить все элементы fields(кроме нулевого) в эту пустую строку, ну и после каждого слова, за исключением последнего, пробел ; с удалением также
```go
if cmd == "добавить" {
	str := ""
	for i := 1; i < len(fields); i++ {
		str += fields[i]
	if i != len(fields)-1 {
		str += " "
	}
	}
	fmt.Println("Вы хотите добавить:", str)
} else if cmd == "удалить" {
	str := ""
	for i := 1; i < len(fields); i++ {
		str += fields[i]
		if i != len(fields)-1 {
			str += " "
		}
	}
	fmt.Println("Вы кажется хотите удалить:", str)
}
```

Также надо, чтобы программа выполнялась, пока не будет команды "выйти",благо это реализуется с помощью бесконечного цикла и команды "break"

```go
for{
	if cmd == "выйти"{
	fmt.Println("до скорого")
	break
}
```

Итоговый вариант программы:
```go
package main

import (
	"bufio"
	"fmt"
	"os"
	"strings"
)

  

func main() {
for {
	scanner := bufio.NewScanner((os.Stdin))
	fmt.Print("Введите команду: ")
	if ok := scanner.Scan(); !ok {
		fmt.Println("Ошибка ввода")
		return
	}
text := scanner.Text()
fields := strings.Fields(text)
if len(fields) == 0 {
	fmt.Println("Вы ничего не ввели")
	return
}
fmt.Println("Команда:", fields[0])
cmd := fields[0]
if cmd == "выйти" {
	fmt.Println("До скорого"
	break
}

if cmd == "добавить" {
	str := ""
	for i := 1; i < len(fields); i++ {
		str += fields[i]
		if i != len(fields)-1 {
			str += " "
		}
	}
	fmt.Println("Вы хотите добавить:", str)
} else if cmd == "удалить" {
	str := ""
	for i := 1; i < len(fields); i++ {
		str += fields[i]
		if i != len(fields)-1 {
			str += " "
		}
	}
	fmt.Println("Вы кажется хотите удалить:", str)
} else if cmd == "help" {
	fmt.Println("Команда: help ")
	fmt.Println("-- выводит список доступных команд")
	fmt.Println("")
	fmt.Println("Команда: добавить {добавить что-от}")
	fmt.Println("-- позволяет что-то добавлять")
	fmt.Println("")
	fmt.Println("Команда: удалить {удалить что-от}")
	fmt.Println("-- позволяет что-то удалить")
} else {
	fmt.Println("Вы ввели неизвестную команду")
	}
}
}
``` 

## Приколдесы, которые изучил в ходе доп практики по материалу этого конспекта
Если у нас идет ветвление по каким-то значениям, по типу
```go
if i == 1{
	// код
}

if i == 14{
 // Код
}

и тд
```
Мы можем это все заменить конструкцией switch(case), пример привел из самостоятельного задания,
```go
switch cmd {

case "съел":
	str := strings.Join(fields[1:], " ")
	foodLog = append(foodLog, str)
	fmt.Println("Записано:", str)
case "список":
	if len(foodLog) == 0 {
		fmt.Println("Дневник пуст")
		return
	}
	fmt.Print("Пользователь съел: ")
	for i, v := range foodLog {
		if len(foodLog)-1 == i {
			fmt.Print(v)
		} else {
			fmt.Print(v + ",")
		}
	}
	fmt.Println("")

default:
	fmt.Println("нет такой команды")
}
```

Мы после слова switch задаем значение, которое должны сравнить с тем, что идет после case, в default указывается то, что выводится,если ни один из case не подошел

Второй приколдес - strings.Join(), вместо того,чтобы писать массив по склеиванию строк, можно использовать этот метод, в качестве первого арнумента идет массив строк, в качестве второго параметр склейки
Пример:
```go
a := []string {"я", "хочу", "спать"}
str := strings.Join(a," ")
fmt.Println(str) // вывод: я хочу спать
```
