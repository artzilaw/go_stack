Переменная из себя представляет некую область в оперативной памяти,в которой хранятся те или иные данные, объявляется так:
```go
a:=0
```

В переменных можно хранить значения любого типа данных:
```go
text := "game over"
score := 111
```

Также они меняются по ходу выполнения программы:

```go
score := 111

  

score = 40

  

fmt.Println(score) //выведет 40
```

## Про типизацию
 
 Go это строго типизированный язык,тоесть задав тип integer,уже через переобъявления значения переменной не получится задать значение иного типа:
```go
text := "game over"

text = 15 // будет ошибка
```

Это нужно,чтобы не тратить время на выяснение того,какой у нас тип

## Арифметика 

Можно выполнять арифметические операции внутри переменной,можно делать выражения вида константа + константа,константа+переменная,переменная+ переменная:

```go
score = 10 + 10 
score = score + 10 
score = score + score 
```

Виды арифметических операций:
```go
score := score + 10
score := score - 15
score := score * 15
score := score / 3
```

## Нюансы с дробной частью   

При выражении:
```go
score := 11
fmt.Println(score / 3) // 3
```

у нас выведется целая часть,но где же дробная? Чтобы вывелась дробная часть,нам нужно,чтобы значение в переменной было типа float,тоесть нам нужно выражение следующего вида:

```go
score := 11.0
fmt.Println(score / 3)
```

## Деление

деление может быть как целочисленным,так и с выводом остатка

целочисленное:
```go
score := 11
fmt.Println(score / 3) // 3
```

с остатком:
```go
score := 11
fmt.Println(score % 3) // 2
```

## Конкатенация строк 

В  языке go мы можем прибавлять строки друг к другу:

```go
text := "hello"
text = text + " world"
fmt.Println(text) // hello world
```

**==Примечание:==** в примере выше без пробела во втором слагаемом будет вывод "helloworld"

## Типы записи в переменной

Есть коротки и длинный вид записи:
```go
score := 3
s := "text"

var number int = 10
var str string = "t"
```

Отличие по сути в том,что у нас в коротком типе записи у переменной должно быть записано какое-то значение,у длинной записи можно не писать значение сразу, это будет считаться объявлением
 В длинный тип записи записывается значение по-умолчанию и для каждого типа данных оно разное(например для integer это 0, а для string пустая строка):
```go 
var number int
var str string

fmt.Println(number) // 0
fmt.Println(str) // ""
``` 

## Инкремент и способы сложения

Инкремент - увеличение значения на единицу
Декремент - уменьшение

Обозначение:
```go
score++
score--
```  

Cпособы сложения(да и не только):
```go
a+=2 // аналог a = a + 2
a-=2 // аналог a = a - 2
```
