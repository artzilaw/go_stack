По ходу знакомства были объяснена прям самая база,про импортируемый модуль "fmt" было пару слов,про то,что запускается код через: 
```go  
go run 'имя файла'
```
также просмотрел отличия между Print и Println(первый не переносит на другую строку,а второй переносит ).
Пример написанного сегодня кода:
```go
package main

  

import "fmt"

  

func main() {

fmt.Println("Hello World") //переносит

fmt.Print("фокус") // не переносит

fmt.Print("покус")

}
```
