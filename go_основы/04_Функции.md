Вообще, одну функцию мы уже используем, это функция main()

```go
package main

import "fmt"

func main() { // вот она
	fmt.Println("Текст")
}
```

Можно ли еще функции создавать? Да, можно, но я буду объяснять ниже что да как

Пример функции:
```go
package main

import "fmt"

func main() {
	fmt.Println("До вызова функции")
	hello()
	fmt.Println("После вызова функции")
}

func hello() {
	fmt.Println("Hello1")
	fmt.Println("Hello2")
}

Вывод:
До вызова функции
Hello1
Hello2
После вызова функции
```

Как же оно работает на примере кода выше? У нас сначала идет работа программы в теле *main()* (вывод "До вызова функции" ), потом идет вызов функции *hello()*, то есть программа перескакивает на функцию *hello()* и выполняет код в ней, когда код выполнился, программа возвращается в *main()*, выводится "После вызова функции" и программа завершается. Функцию можно вызывать несколько раз, тоесть если подряд поставить вызовы функции *hello()*, будет значит несколько раз выводиться "Hello1" и  "Hello2"      

Получается, что функция - это кусок кода,который вызывается 

## Область видимости

В функция переменная доступна только в той функции,где она была создана:

```go
package main

import "fmt"

func main() {
	score := 5 // создали переменную в main()
	fmt.Println(score)
	
	fmt.Println("До вызова функции")
	hello()
	fmt.Println("После вызова функции")
	aboba()
}

func hello() {
	fmt.Println("Hello1")
	fmt.Println("Hello2")
}
func aboba() {
	score = 5 // пытаемся изменить в функции,но тут будет ошибка
	a := 0
	b := 2
	c := 3
	fmt.Println(a + b + c)
}
```

## Аргументы функции

В функции можно передавать аргументы того или иного типа данных,туда можно передавать как литералы,так и переменные

Пример: 

```go
package main

import "fmt"

func main() {
	lenght := 4
	square(lenght) // выведет 16
}

func square(x int) { // передали в качестве аргумента целочисленный x
	fmt.Println(x * x)
}
```

В примере выше функция может принимать только значение int, другие он не может

**==Примечание:==** К слову,очень сильно помогает оптимизировать код и сделать его визуально красивее ,пока автор видео объяснял принципы работы функции,я тут чуть поигрался с таской из видоса тут есть пользовательский ввод,он по видео чуть ли не в конце,но так как я уже учил Go, то что-то помню; ну и возвращение значения из функции, это будет дальше,пока внимания особо не обращайте на это)

```go
package main

import "fmt"

func main() {
	var name string
	name = clientName(name)
	oficiantWork(name)
}

  

func oficiantWork(name string) {
	fmt.Println("Накрываю на стол")
	fmt.Println("Привет,", name)
```

Также наши функции могут приминать несколько аргументов разных типов,а не только один аргумент одного типа:
```go
package main

import "fmt"

func main() {
	sum(1, 2, "Привет")
}

func sum(a int, b int, greeting string) {
	s := a + b
	fmt.Println(greeting)
	fmt.Println("Сумма:", s)
}
```

В примере выше в функции *sum()* аргументы *a* и *b* типа *int*, а аргумент *greeting* типа *string* 

**==Примечание:==** У нас значения в переменную по сути просто копируются. Что это значит? Возьмем пример ниже:
```go
package main

import "fmt"

func main() {
	number := 5
	text := "Привет"
	
	foo(number, text) // вывод того,что в функции
	
	fmt.Println("") 
	
	fmt.Println(number) // вывод 5
	fmt.Println(text) // вывод "Привет"
}

func foo(n int, t string) {
	fmt.Println("n:", n) // выведется 5
	fmt.Println("t:", t) // выведется "Привет"
	
	fmt.Println("")
	
	n = 100
	t = "Текст"
	
	fmt.Println("n:", n) // выведется 100
	fmt.Println("t:", t) // выведется "Текст"
}
``` 

То есть, исходя из работы программы выше, следует то, то функция лишь только копирует значения,которые ей передали в качестве аргумента, но менять их она не может

## Функции,которые возвращают значение

Как было в одном из [[#^90028f | примечаний выше]] , функция может  возвращать значение

```go
package main

import "fmt"

func main() {
	number := sum(1, 2)
	fmt.Println("number:", number)
}

  

func sum(a int, b int) int {
	fmt.Println("a:", a)
	fmt.Println("b:", b)
	s := a + b
	return s
}
```

Что произошло в блоке выше? Была создана функция,ей были переданы аргументы a и b , далее идет какой-то *int*, но  этот int - это тип значения,которое должна вернуть функция,то есть, после того,как эта функция отработала, возвращенное значение должно быть *int* (по примеру выше)
Далее идет описание функции и в конце return; return - это ключевое слово,которое возвращает результату работы функции тот или иной параметр(проще говоря переменную),return может быть в функции только одно и после него команда останавливается 
Для наглядности: в переменную number в функции *main()* было передано в качестве значения результат работы функции *sum()*

Примечание: return может быть несколько при наличии ветвлений, так как как мы можем достичь return только при том или ином условии
```go
package main

import "fmt"

func main() {
	s := str()  
	fmt.Println(s)
}

func str() string {
	subscribed := true
	if subscribed {
		return "Ты подписан на гитхаб"
	} else {
		return "Ты не подписан на гитхаб"
	}
}
```

## return в функции, которая ничего не возвращает

Допустим у нас есть функция,которая приветствует человека по имени, но что если у нас ничего не передается? Тогда надо сделать проверку, но нам надо именно остановить функцию,а в функция,которая ничего не возвращает, выполняется до конца, вот тут то приходит на помощь return 

```go
package main

import "fmt"

func main() {
	greeting("")
}

  

func greeting(name string) {
	if name == "" {
		fmt.Println("Вы передали пустое имя")
		return
	}
	fmt.Println("Привет уважаемый,", name)
}
```

Если бы return  не было, функция бы не прервалась  и у нас вывелось бы "Вы передали пустое имя"  и "Привет уважаемый," вместе

## Глобальные переменные

Представляют из себя переменные,которые будут доступны из всех блоков/функций программы,то есть при неограниченной области видимости(создается вначале и удаляется в конце), мы можем ее допустим менять в одной функции, передавать значение в другой, без необходимости пересоздавать ее  
```go
package main

import "fmt"

var number int = 5 // глобальная переменная

func main() {
	number *= 4 // =20
	greeting("")
	fmt.Println(number) // выведется 10
}

func greeting(name string) {
	number /= 2 // =10
	if name == "" {
		fmt.Println("Вы передали пустое имя")
		return
	}
	fmt.Println("Привет уважаемый,", name)
}
```
